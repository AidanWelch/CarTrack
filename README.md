# CarTrack
An open-source tool to track and identify cars on publicly accessible traffic cameras.

## Build

### Prerequisites 
- CMake
- Ninja
- OpenCV

### Building
1. Open a terminal in `./build`
2. `cmake ..`
3. `../build.sh`
4. `ninja`
5. `./CarTrack`
